# 03 GDD Excerpt

## Core Loop (Draft)
1. Accept weekly order mix.
2. Assign spirits to stations (brew, carve, pack).
3. Resolve simulated production week.
4. Review misses, earn currency, unlock upgrades.
5. Re-plan next week.

## MVP Boundaries
- 3 station types.
- 4 spirit archetypes.
- 10 order templates.
- 1 upgrade tree with 12 nodes.

## Non-Goals (MVP)
- Narrative campaign.
- Multiplayer.
- Real-time character control.
